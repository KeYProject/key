This are the input files for the TimSort case study from 2016

It has been adapted to load in KeY v3.0

Missing at the moment (to be added soon):
- the additional rules for bitshift etc,
- maybe some additional specification of the Java Library Classes


The old proofs are not included as they load only on an older KeY version.
New proofs will be added incrementally.

de Gouw, S., de Boer, F.S., Bubel, R. et al. Verifying OpenJDK’s Sort Method for Generic Collections. J Autom Reasoning 62, 93–126 (2019). https://doi.org/10.1007/s10817-017-9426-4
