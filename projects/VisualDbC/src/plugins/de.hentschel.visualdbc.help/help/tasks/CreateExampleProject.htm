Create an Example Project

This task describes how to create an example project. This is a normal java project that is filled with predefined content.

The following examples exists for the KeY data source:

  • Banking Example
  • KeY Quicktour (for more information visit the KeY website

Execute the following steps to create an example project:

  1. Open New Wizard.
  2. Select "Examples/Visual DbC/Banking Example" or "Examples/Visual DbC/KeY Quicktour".
  3. Select "Next >".
  4. Define a project name like "BankingExample" or "QuicktourExample".
  5. Select "Finish". The project is now created and filled with the example content. An example DbC diagram is also opened.
