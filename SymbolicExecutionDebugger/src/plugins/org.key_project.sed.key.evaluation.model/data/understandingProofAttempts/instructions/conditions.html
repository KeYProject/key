Understanding Proof Attempts

The goal of the evaluation is to compare the inspection of proof attempts using KeY and the Symbolic Execution Debugger (SED). During the evaluation, four proofs and related questions will be shown to you. You can contribute to the evaluation by answering these questions as best as you can. The evaluation usually takes about 60 minutes.

Please read all instructions carefully. You can do nothing wrong as long as you answer all questions as best as you can.

Terms of Use

  • Please participate only if you have not participated before and if you have not seen parts of the evaluation. Otherwise the validity of the evaluation is compromised!
  • You can withdraw from the evaluation at any time without any disadvantage.
  • Collected data is anonymous and you are asked before something will be sent.
  • The collected data will be used for research and publication.
  • An additional project will be created and later deleted during evaluation.
    • Existing projects are not affected by the evaluation.
  • Perform the evaluation alone and not in a group.
Thank you very much!
